File: config.asm
Author: arfy slowy

OVERVIEW
Global constants and macros used throughout the kernel. This file defines fundamental sizes, shifts, masks, and ASCII constants.

PAGE CONSTANTS
  KERNEL_PAGE_mask        = 0xF000 (page mask for 32-bit addressing)
  KERNEL_PAGE_SIZE_byte  = 0x1000 (4096 bytes = 4KB)
  KERNEL_PAGE_SIZE_shift = 12 (2^12 = 4096)

  STATIC_MULTIPLE_BY_64_shift     = 6 (x64)

STRUCTURE: STATIC_STRUCTURE_BLOCK
  Used for linked list nodes in memory allocation
  Size: 4096 bytes (one page) minus 8 bytes for link
  Fields:
    .data: 4088 bytes of usable data
    .link: 8 bytes for next node pointer
