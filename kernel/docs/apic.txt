LOCAL APIC SUPPORT
File: kernel/apic.asm
Author: arfy slowy

OVERVIEW
Provides Local APIC (Advanced Programmable Interrupt Controller) constants,
variables, and helper functions. The Local APIC handles timer interrupts,
inter-processor interrupts, and spurious interrupts.

CONSTANTS
  Register Offsets:
    KERNEL_APIC_ID_register     = 0x0020 (APIC ID)
    KERNEL_APIC_TP_register     = 0x0080 (Task Priority)
    KERNEL_APIC_EOI_register    = 0x00B0 (End of Interrupt)
    KERNEL_APIC_LD_register     = 0x00D0 (Logical Destination)
    KERNEL_APIC_DF_register     = 0x00E0 (Destination Format)
    KERNEL_APIC_SIV_register    = 0x00F0 (Spurious Interrupt Vector)
    KERNEL_APIC_ICL_register    = 0x0300 (Interrupt Command Low)
    KERNEL_APIC_ICH_register    = 0x0310 (Interrupt Command High)
    KERNEL_APIC_LVT_TR_register = 0x0320 (LVT Timer Register)
    KERNEL_APIC_TICR_register   = 0x0380 (Timer Initial Count)
    KERNEL_APIC_TDC_register    = 0x03E0 (Timer Divide Configuration)

  Flags:
    KERNEL_APIC_LD_FLAG_target_cpu        = 0x01000000
    KERNEL_APIC_DF_FLAG_flat_mode         = 0xFFFFFFFF
    KERNEL_APIC_SIV_FLAG_enable_apic      = 0x00000100
    KERNEL_APIC_SIV_FLAG_spurious_vector  = 0x000000FF
    KERNEL_APIC_ICL_COMMAND_COMPLETE_bit  = 12
    KERNEL_APIC_LVT_TR_FLAG_mask_interrupts = 0x00010000

  Timer Divide Values:
    KERNEL_APIC_TDC_divide_by_1   = 0x0B
    KERNEL_APIC_TDC_divide_by_16  = 0x03

  Interrupt Vector:
    KERNEL_APIC_IRQ_number = 0x20 (base for APIC interrupts)

VARIABLES
  kernel_apic_base_address  = qword (MMIO base address, from ACPI)
  kernel_apic_size          = qword (size of MMIO region)
  kernel_apic_count         = byte (number of CPUs found)
  kernel_apic_id_table      = 256 bytes (APIC ID to CPU mapping)

FUNCTIONS

kernel_apic_id_get
  Purpose: Get the APIC ID of the current CPU
  Output: AL = APIC ID
  Uses: macro_apic_id_get (reads from APIC ID register)

REGISTER DETAILS

EOI Register (0x00B0):
  Write any value to signal end of interrupt.
  Must be written in interrupt handlers before iretq.

Spurious Interrupt Vector (0x00F0):
  Bit 8: APIC enable (must be set to enable APIC)
  Bits 0-7: Spurious vector number (typically 0xFF)

LVT Timer Register (0x0320):
  Bits 0-7: Interrupt vector
  Bit 16: Mask interrupts
  Bits 17-18: Timer mode (00=one-shot, 01=periodic, 10=TSC-deadline)

Timer Initial Count (0x0380):
  Initial value for timer countdown.
  Timer fires when count reaches 0, then reloads (if periodic).

Timer Divide Configuration (0x03E0):
  0x03 = divide by 16
  0x0B = divide by 1

NOTES
- Base address is discovered via ACPI (kernel/init/acpi.asm)
- Must enable APIC (bit 8 of SIV register) before use
- EOI must be sent for every interrupt to allow new interrupts
- Timer frequency = bus_frequency / divide_value / TICR_value
