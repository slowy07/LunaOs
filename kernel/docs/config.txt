KERNEL CONFIGURATION
File: kernel/config.asm
Author: arfy slowy

OVERVIEW
This file defines global and kernel configuration constants including kernel metadata, memory layout, and software/service IDs.

FILES
  config.asm            - Global constants (page size, shift/multiply, SOFTWARE_base)
  kernel/config.asm     - Kernel-specific constants (memory high, services)

KERNEL METADATA
  KERNEL_name        = "luna"
  KERNEL_version     = "1"
  KERNEL_revision    = "92"
  KERNEL_architecture = "x86_64"

MEMORY ADDRESSES (kernel/config.asm)
  KERNEL_BASE_address         = 0x0000000000100000 (1MB)
  KERNEL_STACK_address        = Virtual address at top of 128TB virtual space
  KERNEL_STACK_pointer        = KERNEL_STACK_address - 1 page
  KERNEL_STACK_SIZE_byte      = 2 * KERNEL_PAGE_SIZE_byte (8KB)
  KERNEL_STACK_TEMPORARY_pointer = 0x8000 + 4KB (temporary stack for AP boot)
  KERNEL_MEMORY_HIGH_mask     = 0xFFFF000000000000 (upper half canonical mask)
  KERNEL_MEMORY_HIGH_REAL_address = 0xFFFF800000000000
  KERNEL_MEMORY_HIGH_VIRTUAL_address = KERNEL_MEMORY_HIGH_REAL_address - KERNEL_MEMORY_HIGH_mask

SOFTWARE (config.asm)
  SOFTWARE_base_address = KERNEL_MEMORY_HIGH_REAL_address

SERVICE IDS (kernel/config.asm)
  KERNEL_SERVICE           = 0x40 (interrupt vector for syscall gate)
  KERNEL_SERVICE_PROCESS   = 0x00
  KERNEL_SERVICE_PROCESS_exit = 0x0000
  KERNEL_SERVICE_VIDEO     = 0x01
  KERNEL_SERVICE_VIDEO_string = 0x0100
