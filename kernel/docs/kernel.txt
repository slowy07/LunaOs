KERNEL MAIN
File: kernel/kernel.asm
Author: arfy slowy

OVERVIEW
The main kernel file that ties together all components. This is the entry point after bootloader transfers control.

STRUCTURE

[ORG KERNEL_BASE_address] = 0x100000 (1MB)

32-bit entry point:
  init:
    Includes kernel/init.asm (main init chain)

64-bit kernel:
  kernel:
    Infinite loop (jmp $)
  
  Then includes all subsystems in order:
    Macros:
      - kernel/macro/close.asm: Semaphore lock macro
      - kernel/macro/apic.asm: APIC ID macro
    
    Kernel core:
      - kernel/panic.asm: Error/panic handler
      - kernel/page.asm: Page table management
      - kernel/memory.asm: Physical memory allocation
      - kernel/video.asm: VGA text output
      - kernel/apic.asm: Local APIC support
      - kernel/io_apic.asm: IO-APIC support
      - kernel/data.asm: Data structures (GDT, TSS, IDT headers)
      - kernel/idt.asm: Interrupt handling
      - kernel/task.asm: Task scheduler
      - kernel/vfs.asm: Virtual File System
    
Drivers:
  - kernel/driver/rtc.asm: Real-time clock driver
  - kernel/driver/ps2.asm: PS/2 keyboard/mouse driver
  - kernel/driver/storage/ide.asm: IDE ATA/ATAPI storage driver
    
    Libraries:
      - library/page_align_up.asm: Page alignment utility
      - library/page_from_size.asm: Size to page conversion

EXECUTION FLOW

1. Bootloader loads kernel at 0x100000
2. Bootloader validates Multiboot header
3. Jumps to init (32-bit)
4. Init calls long_mode.asm to enable 64-bit
5. After enabling paging, continues with other init modules
6. Init completes, jumps to kernel: (halt loop)

LOAD ORDER

Configuration:
  config.asm                    (global config)
  kernel/config.asm             (kernel config)

Init (32-bit):
  kernel/init.asm               (includes all kernel/init/*.asm)
  kernel/init/font.asm          (font name display, between video and memory)
  kernel/init/storage.asm       (storage subsystem stub, between vfs and network)
  kernel/init/network.asm       (NIC detection, between storage and task)

Macros:
  kernel/macro/close.asm        (semaphore lock)
  kernel/macro/apic.asm         (APIC ID)

Kernel Core (64-bit):
  kernel/panic.asm
  kernel/page.asm
  kernel/memory.asm
  kernel/video.asm
  kernel/apic.asm
  kernel/io_apic.asm
  kernel/data.asm
  kernel/idt.asm
  kernel/task.asm
  ; kernel/thread.asm       (commented out for testing)
  kernel/vfs.asm
  kernel/exec.asm           (process exec support)
  kernel/service.asm        (syscall dispatcher, vector 0x40)
  kernel/font/setfonts.asm

Drivers:
  kernel/driver/rtc.asm
  kernel/driver/ps2.asm
  kernel/driver/storage/ide.asm

Services:
  kernel/service/shell.asm
  kernel/service/http.asm
  kernel/service/tx.asm
  kernel/service/network.asm     (includes sub-files: config, data, checksum, arp, icmp, tcp, wrap)

Libraries:
  library/page_align_up.asm          - Page alignment
  library/page_from_size.asm         - Size to page conversion
  library/string_compare.asm         - String comparison
  library/string_cut.asm             - Cut string at delimiter
  library/string_digits.asm          - Validate digit characters
  library/string_to_integer.asm      - Convert string to integer
  library/string_trim.asm            - Trim whitespace
  library/string_word_next.asm       - Extract next word
