MACRO: CLOSE (Semaphore Lock)
File: kernel/macro/close.asm
Author: arfy slowy

MACRO: macro_close

PURPOSE
Acquire a binary semaphore (spinlock) using atomic test-and-set.

USAGE
  macro_close variable, offset

PARAMETERS
  %1: Base address of semaphore variable
  %2: Offset into the variable (for array access)

IMPLEMENTATION
  Uses LOCK XCHG instruction for atomic compare-and-swap
  
  Algorithm:
    1. Move STATIC_TRUE (1) into AL
    2. Atomically exchange AL with [variable + offset]
    3. If AL was non-zero (lock was held), retry
    4. Otherwise, lock acquired

PSEUDO-CODE
  do {
    al = 1;
    old = atomic_xchg(&semaphore[offset], al);
  } while (old != 0);

REGISTERS
  Altered: RAX (saved and restored)
  Preserves: All other registers

EXAMPLE
  macro_close kernel_memory_lock_semaphore, 0
  
  Acquires the memory allocator lock.

NOTE
This is a busy-wait (spinlock) semaphore.
Should only be held for very short periods.
Must be released by setting the byte to 0.
