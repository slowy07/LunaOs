RTC DRIVER
File: kernel/driver/rtc.asm
Author: arfy slowy

OVERVIEW
Real Time Clock driver that provides periodic interrupts at 1024Hz.
Also exposes date/time readout via driver_rtc_get_date_and_time.

STRUCT RENAMES (rev 92)
  DRIVER_RTC_STRUCTURE_TIME -> DRIVER_RTC_STRUCTURE
  driver_rtc_time -> driver_rtc_date_and_time

VARIABLES
  driver_rtc_semaphore   db  (STATIC_FALSE)
  driver_rtc_microtime   dq  (microsecond counter, incremented each interrupt)
  driver_rtc_date_and_time dq  (current date/time fields via STRUCTURE)

PROCEDURES

driver_rtc (ISR):
  Increments driver_rtc_microtime
  Reads status register C (dummy read to clear interrupt)
  Sends EOI to APIC

driver_rtc_get_date_and_time:
  Reads CMOS ports for second, minute, hour
  Stores in driver_rtc_date_and_time struct
