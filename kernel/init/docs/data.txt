INIT DATA - STATIC DEFINITIONS
File: kernel/init/data.asm
Author: arfy slowy

OVERVIEW
Central data repository for initialization: display strings, error
messages, semaphores, VFS directory structure, preloaded file
binaries, and the kernel services startup list.

VARIABLE LAYOUT (in assembly order)

+--------------------------------------------+
| Display strings (null-terminated, with _end)|
| - welcome, video, font, memory, storage    |
+--------------------------------------------+
| Error message strings                      |
| - memory_low, acpi, acpi_corrupted,        |
|   apic, ioapic                             |
+--------------------------------------------+
| Status message strings                     |
| - acpi_search, acpi_search_found           |
+--------------------------------------------+
| Semaphores (5 bytes)                       |
| - apic, ioapic, smp, ap, global (data.asm) |
+--------------------------------------------+
| Counters (2 bytes)                         |
| - ap_count, apic_id_highest                |
+--------------------------------------------+
| Services list                              |
| - tresher, tx, network, HTTP               |
+--------------------------------------------+
| VFS directory structure                    |
| - [/bin, 4 bytes] [/dev, 4 bytes] [0]     |
+--------------------------------------------+
| VFS files table                            |
| - 3 entries + terminator                   |
|   [/bin/init] [/bin/shell] [/bin/wello]   |
+--------------------------------------------+
| Embedded binaries                          |
| - init (incbin "build/init")              |
| - shell (incbin "build/shell")            |
| - wello (incbin "build/wello")            |
+--------------------------------------------+
| Boot trampoline                            |
| - boot (incbin "build/boot")              |
+--------------------------------------------+

SEMAPHORE USAGE

kernel_init_apic_semaphore   -- Lock for APIC MMIO register access
kernel_init_ioapic_semaphore -- Lock for IO-APIC register access
kernel_init_smp_semaphore    -- BSP vs AP path selector (FALSE=BSP, TRUE=AP)
kernel_init_ap_semaphore     -- AP boot synchronization barrier
kernel_init_semaphore        -- (in kernel/data.asm) Global init done flag

SERVICES LIST (kernel_init_services_list)

List format: array of dq pointers, terminated by dq STATIC_EMPTY

Order  Module         Function
-----  ------         --------
  1    service_tresher  Task reaper (reclaims closed task slots)

Services run as kernel tasks. Started sequentially in init/services.asm.
Note: service_tx, service_network, and service_http were removed from
the boot-time services list. Only tresher is started at boot.

VFS DIRECTORY STRUCTURE (kernel_init_vfs_directory_structure)

Format: [length_byte][path_bytes] per entry, 0x00 terminator

Entry 1: db 0x04, "/bin"     -> creates /bin directory
Entry 2: db 0x04, "/dev"     -> creates /dev directory
End:     db STATIC_EMPTY

VFS FILES TABLE (kernel_init_vfs_files)

Format per entry (KERNEL_INIT_STRUCTURE_VFS_FILE):
+0x00: .data_pointer qword  -- pointer to embedded binary data
+0x08: .size         qword  -- size of binary data in bytes
+0x10: .length       byte   -- length of path string
+0x11: .path         var    -- path string (e.g. "/bin/init")
Entry size: .SIZE = 17 + path_length

Table terminated by dq STATIC_EMPTY

Entry  Path           Binary                 Source
-----  ----           ------                 ------
  1    /bin/init      kernel_init_vfs_file_init  incbin "build/init"
  2    /bin/shell     kernel_init_vfs_file_shell incbin "build/shell"
  3    /bin/wello     kernel_init_vfs_file_hello incbin "build/wello"

EMBEDDED BINARY: kernel_init_boot_file
  incbin "build/boot" -- SMP AP boot trampoline
  Loaded at 0x8000 during SMP init. Contains 16-bit real-mode code
  that transitions APs to 64-bit long mode.
