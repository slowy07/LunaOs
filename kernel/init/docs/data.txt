VARIABLES
Semaphores (Mutex Locks):
  | Variable              | Type     | Purpose           
  | kernel_init_apic_semaphore | byte  | Lock for APIC operations
  | kernel_init_ioapic_semaphore| byte | Lock for IO-APIC operations
  | kernel_init_smp_semaphore   | byte | Lock for SMP operations
  | kernel_init_ap_semaphore    | byte | AP boot synchronization
  | kernel_init_semaphore       | byte | Global init complete flag (kernel/data.asm)

Counters:
  | Variable              | Type     | Purpose
  | kernel_init_ap_count  | byte     | Number of Application Processors (APs))
  | kernel_init_apic_id_highest | byte | Highest APIC ID detected

VFS Directory Structure:
  | Variable              | Purpose
  | kernel_init_vfs_directory_structure | Predefined directory list for VFS root
  |                       | Format: [length_byte][path_string] per entry, 0x00 terminator
  |                       | Current entries: /dev

Embedded Binary:
  | Variable              | Purpose
  | kernel_init_boot_file | SMP AP boot trampoline (incbin of boot.asm)
  | kernel_init_boot_file_end | End marker for size calculation

Service List (kernel_init_services_list):
  1. service_tresher  - Task reaper (cleans up closed tasks)
  2. service_tx       - Packet transmission service
  3. service_network  - Network stack (IPv4/TCP/UDP/ICMP/ARP)
  4. service_http     - HTTP server (port 80, IPC receive loop)
  5. service_shell    - Interactive shell
  Terminator: dq STATIC_EMPTY

  Services are started sequentially during init. Each registers its
  own PID via kernel_task_active on first execution.

Constants Used:
  - STATIC_FALSE        : 0 (unlocked/disabled)
  - STATIC_TRUE         : 1 (locked/enabled)
  - STATIC_EMPTY        : 0
