IPC - INITIALIZATION
File: kernel/init/ipc.asm
Author: arfy slowy

OVERVIEW
Allocates the initial IPC message pool page during kernel boot.
Runs after PS/2 init and before network/task init, so that the
IPC subsystem is available for service task communication.

PROCEDURE

kernel_init_ipc:
  Allocates one page (KERNEL_IPC_SIZE_page_default) for the IPC pool.
  Calls kernel_memory_alloc to reserve the page.
  Calls kernel_page_drain_few to populate the page table.
  Stores the page address in kernel_ipc_base_address.
  Sets the first block's link to itself (circular single-entry pool).
