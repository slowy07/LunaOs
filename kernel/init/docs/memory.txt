MEMORY INITIALIZATION
File: kernel/init/memory.asm
Author: arfy slowy
Date: 2026
License: MIT

OVERVIEW
Parses Multiboot memory map, initializes page bitmap allocator, and sets up
kernel memory management.

KEY VARIABLES
  kernel_page_total_count    : Total pages available
  kernel_page_free_count    : Free pages (set to total at init)
  kernel_memory_map_address : Bitmap start address
  kernel_memory_map_address_end : Bitmap end address

BITMAP FORMAT
  Bit = 0 : Page is allocated
  Bit = 1 : Page is free
