This module initializes the Real Time Clock (RTC) hardware, configuring it to
generate periodic interrupts at 1024Hz and setting up the IDT entry for RTC
interrupt handling.

FUNCTION: kernel_init_rtc

Purpose: Initialize RTC hardware and connect to interrupt system

Flow Diagram:

+----------+-------------+

+------------------------+

+------------------------+

Hardware Configuration:
  - Interrupt Rate: 1024 Hz (configurable via rate bits)
  - Time Format: 24-hour, binary (not BCD)
  - Interrupt Mode: Periodic interrupts enabled

I/O Ports:
  - Command Port: 0x70 (CMOS address register)
  - Data Port: 0x71 (CMOS data register)

Status Register A (0x0A):
  - Rate: 0x06 (1024 Hz)
  - Divider: 0x20 (select 32.768KHz clock)

Status Register B (0x0B):
  - 24-hour mode: 0x02
  - Binary mode: 0x04
  - Periodic interrupt: 0x40

IRQ Configuration:
  - IRQ Number: 8
  - IDT Vector: 0x28 (IRQ base 0x20 + IRQ 8)
  - IO-APIC Register: KERNEL_IO_APIC_iowin + (8 * 2)

Dependencies:
  - kernel_idt_mount        : Register ISR in IDT
  - kernel_io_apic_connect  : Connect IRQ to IO-APIC
