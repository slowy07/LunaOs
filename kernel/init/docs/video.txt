This module initializes the video/display subsystem by reading framebuffer
information from the Multiboot header and clearing the screen.
Purpose: Initialize framebuffer video output and display welcome message

Flow Diagram:

+------------------------+
| Start kernel_init_video|
+----------+-------------+
           |
           v
+------------------------+
| push rbx               |
+----------+-------------+
           |
           v
+------------------------+
| Check multiboot flags  |
| bit 12 (video flag)    |
+----------+-------------+
           |
           v
+------------------------+
| Read framebuffer addr  |
| from MULTIBOOT_HEADER  |
+----------+-------------+
           |
           v
+------------------------+
| kernel_video_base_addr |
| = framebuffer_base     |
| kernel_video_pointer   |
| = framebuffer_base     |
+----------+-------------+
           |
           v
+------------------------+
| Read framebuffer       |
| width and height       |
+----------+-------------+
           |
           v
+------------------------+
| Store width x height   |
| in kernel_video_       |
| size_pixel             |
+----------+-------------+
           |
           v
+------------------------+
| Convert to bytes:      |
| kernel_video_size_byte |
+----------+-------------+
           |
           v
+------------------------+
| Calculate char dims:   |
| kernel_video_width_char|
| kernel_video_height_char
+----------+-------------+
           |
           v
+------------------------+
| Calculate scanline_byte|
| and scanline_char      |
+----------+-------------+
           |
           v
+------------------------+
| Clear screen           |
| --> kernel_video_drain |
+----------+-------------+
           |
           v
+------------------------+
| Print banner:          |
| "Luna Operating System |
|  (v1.90 x86_64)"       |
+----------+-------------+
           |
           v
+------------------------+
| Print ":: Video        |
|  resolution "          |
+----------+-------------+
           |
           v
+------------------------+
| Print width (decimal)  |
+----------+-------------+
           |
           v
+------------------------+
| Print "x" separator    |
+----------+-------------+
           |
           v
+------------------------+
| Print height (decimal) |
+----------+-------------+
           |
           v
+------------------------+
| Print newline          |
+----------+-------------+
           |
           v
+------------------------+
| pop rbx                |
| Return                 |
+------------------------+

Output:
  - "Luna Operating System (v1.90 x86_64)"
  - ":: Video resolution <width>x<height>"
  - Displayed on LFB framebuffer via kernel_video_string/kernel_video_number

Dependencies:
  - kernel_video_drain   : Clear/initialize framebuffer with background color
  - kernel_video_string  : Print null-terminated string with color support

Framebuffer:
  - Address from MULTIBOOT_HEADER.framebuffer_addr
  - 32 bits per pixel (KERNEL_VIDEO_DEPTH_bit = 32)
  - Width/height from MULTIBOOT_HEADER.framebuffer_width/height
    (set by bootloader, configured via -dMULTIBOOT_VIDEO_WIDTH/HEIGHT_pixel)
  - Character cell size: FONT_WIDTH x FONT_HEIGHT pixels
  - Computed vars: size_pixel, size_byte, width_char, height_char,
    scanline_byte, scanline_char stored to kernel_video_* globals
