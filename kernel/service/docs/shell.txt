SHELL SERVICE
File: kernel/service/shell.asm
Author: arfy slowy

OVERVIEW
Text-based interactive shell. Displays a prompt, reads user input,
trims whitespace, extracts the first word, and dispatches to the
prompt handler.

CONSTANTS (from shell/config.asm)
SERVICE_SHELL_CACHE_SIZE_byte = KERNEL_VIDEO_WIDTH_char - prompt_len - 1

PROCEDURES

service_shell:
  1. Wait for kernel_init_semaphore == STATIC_FALSE (init complete)
  2. Draw prompt string (with or without leading newline based on cursor)
  3. Clear EBX (word buffer index)
  4. Read input via library_input with SERVICE_SHELL_CACHE_SIZE_byte limit
  5. Trim input, extract first word into EBX
  6. Jump to service_shell_prompt for command dispatch
  7. service_shell_prompt returns by jumping back to service_shell.main

LABELS:
  service_shell    - Entry point (includes init semaphore wait loop)
  .main            - Prompt + input loop body
  .prompt          - Draw prompt and fall through to .restart
  .restart         - Read input, trim, extract word, dispatch

INCLUDES (included inline)
  shell/config.asm  - Cache size constant
  shell/data.asm    - Prompt strings, command strings
  shell/prompt.asm  - Command dispatch logic
