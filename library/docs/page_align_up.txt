LIBRARY: PAGE ALIGN UP
File: library/page_align_up.asm
Author: arfy slowy

FUNCTION: library_page_align_up

PURPOSE
Align a size up to the next page boundary.

INPUT
  Stack (pushed by caller): Size in bytes (qword)

OUTPUT
  RDI: Aligned size (rounded up to page boundary)

ALGORITHM
  1. Get size from stack
  2. AND with page mask (0xFFF) to get offset within page
  3. If offset is 0, size is already aligned -> return 0
  4. Otherwise, calculate: page_size - offset
  5. Add to original size

PAGE MASK
  KERNEL_PAGE_mask = ~0xFFF (negates to 0x...FFFFF000)
  
  This masks off the lower 12 bits (page offset).

EXAMPLE
  Input: 0x1234 (decimal 4660)
  Page align: 0x1000 (4096)
  Result: 0x2000 (8192)
  
  Input: 0x1000 (4096) - already aligned
  Result: 0 (no addition needed)

NOTE
This is a helper function for page allocation.
Used to calculate how many pages are needed for a given size.
