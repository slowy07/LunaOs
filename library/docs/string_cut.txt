STRING CUT
File: library/string_cut.asm
Author: arfy slowy

OVERVIEW
Searches a string for a delimiter character and returns how many
characters were consumed before the delimiter (or the full length
if not found).

PROCEDURE

library_string_cut:
  Input:  AL  = delimiter character (e.g. STATIC_ASCII_DOT)
          RSI = string pointer
          RCX = max length to search
  Output: Stack[0] updated: original RCX - characters remaining
          (the difference is the number of chars before delimiter)
  Notes:  Stops at STATIC_ASCII_TERMINATOR or when delimiter is found.
          If no delimiter found, returns the full length.
