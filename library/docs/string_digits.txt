STRING DIGITS
File: library/string_digits.asm
Author: arfy slowy

OVERVIEW
Validates that every byte in a buffer is an ASCII digit (0x30-0x39).

PROCEDURE

library_string_digits:
  Input:  RSI = string pointer
          RCX = length to check
  Output: CF = clear if all characters are digits
          CF = set if any character is not a digit
  Notes:  Preserves RSI and RCX.
